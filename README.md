# 67kkit Command Blocker

A simple Nukkit plugin to hide server version information.

## Features
* Blocks `/about`, `/ver`, and `/version`.
* Works on everyone, including **Operators (OP)**.
* Custom 2-line colored message response.

## Installation
1. Download the compiled `.jar`.
2. Place it in your Nukkit `plugins` folder.
3. Restart the server.
